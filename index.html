IDP Linux Terminal Challenge v1.0

Bienvenido al simulador interactivo de comandos para la asignatura de Hardware y Software de la carrera de Telecomunicaciones.

Este sistema evaluará tus conocimientos prácticos basándose estrictamente en las guías de laboratorio de comandos básicos, entornos y administración de particiones primarias.

Reglas del Juego:

  • El banco completo contiene 20 situaciones reales, pero el sistema seleccionará 10 desafíos aleatorios para esta sesión. ¡Cada intento será diferente!
  • Debes ingresar el comando de consola exacto que resuelve la situación planteada.
  • El sistema limpiará automáticamente espacios innecesarios antes y después de tu respuesta.
  • Realimentación Instantánea: Si te equivocas, el sistema te explicará inmediatamente el error conceptual y la forma correcta de solucionarlo para fijar el aprendizaje.
  • Al finalizar, se calculará tu porcentaje de logro y tu Nota Final bajo la escala estándar con un 60% de exigencia para la aprobación (Nota 4.0).
